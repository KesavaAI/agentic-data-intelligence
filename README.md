# ABB Enterprise Agentic Analytics Hub

An enterprise multi-agent analytics pipeline utilizing **LangGraph** and Open-Source LLMs to parse, filter, and dynamically visualize industrial transactional data from workbooks without vulnerable SQL string compilation.

## Quickstart Guide
1. Create a local environment and run configuration setup: `pip install -r requirements.txt`
2. Populate the local `.env` file with your authorized `GROQ_API_KEY`.
3. Construct the spreadsheet: `python generate_data.py`
4. Launch the application portal: `streamlit run app.py`
